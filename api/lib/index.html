DeepIndex

DeepIndex: [number, number][]
  • Defined in packages/core/src/types.ts:34

A deep index is a 3d coordinate. It is a list of 2d vector coordinates. Each item in the list represents a 2d coordinate at a particular level of the 3d nested structure.

DeepOrderingNode

DeepOrderingNode: { children: DeepOrderingNode[][]; disabled: boolean; key: string | null }
  • Defined in packages/core/src/types.ts:9

Represents the underlying selectable tree contained within a particular DOM structure. The selectable tree may be sparser than the actual DOM tree. Its shape may not even be equivalent. For instance, the ordering tree nodes have a 2d array of children, whereas DOM can only have 1d child arrays. Interpreting DOM structure into an ordering tree is done in ./selection.ts

Type declaration

  • children: DeepOrderingNode[][]

    Children are stored as a list of rows to create a 2d structure. To index into children, use the Y index first to select the desired row, then the X index to index into that row.

  • disabled: boolean

    Whether the element should be skipped in ordering

  • key: string | null

    The dimension selectable key of the node represented at this part of the tree. null means this is the root, which is a virtual node that holds the collection of initially selectable sibling children.
